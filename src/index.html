Using Offsets

Using offsets we can move content to the left without creating a column div element. At the 768px breakpoint all columns will default to full width and the offset will disappear

elr-col-4 elr-col-offset-4
elr-col-3 elr-col-offset-8

Responsive Grid Columns

By default all columns are full width on windows smaller than 768px. This is not always what we want. Sometimes we want to control how columns are displayed on specific devices.

We may want a section to have half columns on mobile devices. For this we can use responsive grid columns.

These columns will resize as the screen width changes. But they will never take up more than half of their container
